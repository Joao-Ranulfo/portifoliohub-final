# Política de Segurança - PortfolioHUB

## Versões Suportadas
Apenas a versão mais recente da implantação recebe atualizações de segurança.

| Versão | Suportada |
| :--- | :--- |
| 1.0.0 |  Sim |

## Como relatar uma vulnerabilidade
Se você encontrar qualquer falha de segurança ou vazamento de dados neste portfólio, por favor, NÃO abra uma issue pública. 

Envie um e-mail diretamente para o administrador em: *joao.rvaz@sempreceub.com* com os detalhes do problema. Responderemos em até 48 horas para corrigir a falha.
